About AgilePredict

AgilePredict forecasts wholesale day-ahead electricity prices and converts those forecasts into Octopus Agile import prices for each supported region. The raw day-ahead price is also available as Region Z and is shown in £/MWh.

The model is trained on historical prices, demand, generation forecasts, and selected weather inputs. Current forecasts are generated with a regularised XGBoost DART regressor. Experimental outputs from a classified plunge/normal model and an Extra Trees regressor are stored alongside the main forecast for analysis, but the published Agile price forecast uses the main XGBoost day-ahead prediction.

Current Model Inputs

Feature Description Source
bm_wind Forecast metered wind generation Elexon BMRS / NESO
solar Embedded solar generation forecast NESO
emb_wind Embedded wind generation forecast NESO
demand Forecast national electricity demand Elexon BMRS / NESO
peak Flag for the 16:00 to 19:00 peak pricing window Derived
days_ago Age of the forecast when making the prediction Derived
weekend Weekend indicator Derived

The production feature set is generation-focused.

How Forecasts Are Built

  1. Recent forecasts and actual prices are loaded from the local database.
  2. Training rows are selected from the PM auction window, aligned around the 16:15 update.
  3. The model predicts day-ahead prices for future half-hour slots.
  4. Predictions are blended with known actual prices where appropriate.
  5. Regional Agile prices are derived from the day-ahead forecast using the configured region scale and shift factors.
  6. Low and high forecast bands are generated for the most recent forecast.

Comparison Forecasts

The History page compares AgilePredict against stored external forecasts from AgileForecast and X2R. X2R does not publish a direct National Average region, so the Region X comparison is inferred from Region G using the same configured scale and shift logic used elsewhere in AgilePredict.

External forecasts are for comparison only. They do not feed into the published AgilePredict model.

Updates and Accuracy

Forecasts are updated automatically during the day, with the main update aligned to the day-ahead auction cycle. Accuracy is usually better for broad price shape and daily level than for exact half-hour prices, especially further into the forecast horizon.

AgilePredict is an open-source forecasting project. Please use the forecasts with care and check official pricing when the decision matters.
